LLVM REFERENCES

This file collects high-value references for learning LLVM from fundamentals to advanced topics.

HOW TO USE THIS FILE
1. Start with OFFICIAL LLVM DOCUMENTATION and OFFICIAL TUTORIALS.
2. Use STACK OVERFLOW and STACK EXCHANGE for specific blockers.
3. Use UNIVERSITY MATERIAL for deeper theory and assignments.
4. Use OPEN SOURCE ARTICLES for practical engineering context.
5. Revisit this list per roadmap topic instead of reading everything at once.

OFFICIAL LLVM DOCUMENTATION
1. LLVM Project Home: https://llvm.org/
2. LLVM Documentation Index: https://llvm.org/docs/
3. LLVM Language Reference Manual (IR): https://llvm.org/docs/LangRef.html
4. LLVM Command Guide: https://llvm.org/docs/CommandGuide/
5. Clang Documentation: https://clang.llvm.org/docs/
6. LLVM Doxygen API Docs: https://llvm.org/doxygen/
7. LLVM Programmer's Manual: https://llvm.org/docs/ProgrammersManual.html
8. LLVM Coding Standards: https://llvm.org/docs/CodingStandards.html
9. LLVM Getting Started: https://llvm.org/docs/GettingStarted.html

OFFICIAL TUTORIALS
1. Kaleidoscope Tutorial: https://llvm.org/docs/tutorial/
2. New Pass Manager (Writing Passes): https://llvm.org/docs/WritingAnLLVMNewPMPass.html
3. TableGen Fundamentals: https://llvm.org/docs/TableGen/

RECOMMENDED BOOKS

2. Getting Started with LLVM Core Libraries by Bruno Cardoso Lopes and Rafael Auler
3. LLVM Essentials by Suyog Sarda and Mayur Pandey

Tips for Q/A platforms:
1. Include exact clang/opt/llc version in questions.
2. Provide a minimal reproducer (.c or .ll file).
3. Include exact command and error output.
4. Mention target triple when backend behavior differs.
