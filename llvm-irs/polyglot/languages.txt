# LLVM Frontends: A Polyglot World

One of LLVM's biggest strengths is that many different languages compile down to the same IR.

## 1. Rust (`rustc`)
Rust uses LLVM as its primary backend.
- **Command:** `rustc --emit=llvm-ir source.rs`
- **Characteristics:** IR often contains "noalias" parameter attributes (for borrow checker guarantees) and bounds checks.

## 2. Swift (`swiftc`)
Swift also uses LLVM.
- **Command:** `swiftc -emit-ir source.swift`
- **Characteristics:** Uses a lot of Swift-specific calling conventions (`swiftcc`) and ARC (Automatic Reference Counting) intrinsics.

## 3. Julia
Julia uses LLVM JIT (Just-In-Time) compiler.
- **Inspection:** In the Julia REPL, use `@code_llvm my_function()`.

## 4. Kotlin Native
Kotlin Native compiles Kotlin code to LLVM bitcode to produce standalone binaries without the JVM.

## 5. Scripting Languages (JIT)
- **Python (Numba):** Uses LLVM to JIT compile numerical Python code.
- **Ruby (Rubinius):** Historically used LLVM.
